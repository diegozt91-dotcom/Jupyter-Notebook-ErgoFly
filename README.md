Este programa ha sido diseñado para el cálculo atomatizado de las métricas de electrorretinogramas para parámetros como el 'On', el 'Off', la 'Amplitud de Receptor' o el 'Overshotting'. 
Los resultados obtenidos en este análisis han sido publicados en la revista PlosOne en el siguiente enlace https://doi.org/10.1371/journal.pone.0282348
